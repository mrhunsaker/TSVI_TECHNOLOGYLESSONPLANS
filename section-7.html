Section 1.7 Calculator

The BrailleNote has a calculator application. Though the calculator is scientific, beginners can use it for basic addition, subtraction, multiplication, and division.

Subsection 1.7.1 Tasks

  1. Set UEB Code

    1. Start Scientific Calculator

      • From the Main Menu letter s

      • From anywhere backspace + enter + s

    2. Go to the Option Menu space + o

    3. Braille Settings letter b

    4. Select UEB as the braille code letter e

  2. Explore Calculator Functions

    1. Open calculator from Main Menu letter s

    2. Write a simple math problem

    3. For addition + = dots5,235

    4. For subtraction minus = dots5,36

    5. For multiplication x = dots5,236

    6. For division \textdiv = dots5,34

    7. For a decimal point . = dots256.

    8. To get the result enter

    9. To clear the calculation

      • Clear the calculation backspace + dots14

      • Clear the calculationSpace + dots356

      • Clear the calculationScroll wheel down arrow

    10. Practice with math problems at your level

    11. Delete the last character written backspace

    12. To hear the last number or operator space + dots25.

    13. Alternate displaying the result or the whole calculation

      • Clear the calculation previous + next thumb-keys

      • Clear the calculation scroll wheel left, right arrow

    14. To hear more commands, use context sensitive help press space + h

    15. Practice as time.
