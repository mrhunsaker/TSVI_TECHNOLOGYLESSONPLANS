Section 2.7 User's Guide

Skills Addressed in this Lesson:

  • Enter the User Guide through the Options Menu.

  • Read the table of contents within the User Guide.

  • From within the User Guide, enter the index.

  • Access an item from the table of contents.

  • Access an item from the index options.

  • Read the items in the User Guide as desired.

  • Follow the BrailleNote prompts in the User Guide as necessary.

  • Exit the User Guide.

Subsection 2.7.1 Tasks

  1. Instruct your student to enter the User Guide. He or she should do this by first going to the options menu with space and o. Then press a u to enter the User Guide.

  2. Your student may be given a cue such as "continue reading alarm?" Press an n for no.

  3. A cue such as "press t for table of contents or i for index" should be displayed. Your student should press an i for index.

  4. Read through the items in the index by using space. Use backspace to move back by item. The back and advance thumb-keys will also move back and forward by index item.

  5. Explore the index items. The initial letter of an item can be used to jump directly to an index item.

  6. Move to the index item called "battery" and press enter. The following commands can be used to read when in the User Guide index. Practice reading about the battery.

    1. To start auto-advance, press space with dots1 2 4 5 6.

    2. To start continuous speech, press space with g.

    3. To return to the index, press backspace.

    4. Space with e will exit the User Guide.

    5. To move by subsection, press space with dots4 5 6 or space with dots1 2 3.

    6. Space with dot1 and space with dot4 will move by sentence.

    7. To move by paragraph, use space with dots5 6 and space with dots2 3.

  7. Exit the User Guide. Return to the Main Menu.

  8. From Main Menu, open the User Guide. At the cue, continue reading about the battery, press n for no.

  9. At the cue, press t for table of contents. Use the commands below to read through the table of contents items.

    1. Space or backspace will move forward and back by item.

    2. Space with e will return to the table of contents.

    3. Enter on the item you wish to read.

  10. Go to the word processor chapter and press enter. Sub-index items will then be given.

  11. Move to and from sub-indexes in the usual manner. Press enter on a sub-index item you would like to read.

  12. Your student may use the same commands to read the table of contents as he or she did to read the index.

  13. When your student is finished, return to the Main Menu and turn the BrailleNote off. Provide time for practice and exploration within the User Guide because it's a great way to learn!
