Section 1.1 Create a New Document

Subsection 1.1.1 Tasks

  1. Create a New Document

    1. Turn on the BrailleNote on/off rocker switch

    2. Enter Main Menu space + dots123456

    3. Change Speech mode Previous Thumb-key + space

    4. From Main Menu, open the Word Processor letter w

    5. Create a document letter c

    6. For list of folders space

    7. Scroll items in list space

    8. Open folder enter

    9. Writing a name of the new document (can use contracted UEB)

    10. Create document enter

    11. Delete last character written backspace

    12. Exit current program and return to menu space + e

    13. There is no need to save the document. This is automatically done for you.

    14. Turn off the BrailleNote rocker switch on left side

    15. Repeat the lesson to mastery

  2. Using the Context Dependent Help Menu

    • Review the commands available space + h.

    • Move forward in list of options space

    • Move backward in list of options backspace

    • Select or open a submenu or option enter

Subsection 1.1.2 Quiz

If the statement is true, write or say "true". Write or say "false" if the statement is false.

  1. Space + dots123456 is a quick way to return to Main Menu.

  2. From Main Menu, letter w will open the Word Processor.

  3. The General folder is on a BrailleNote when you get it.

  4. Space in a document to delete the last character written.

  5. The BrailleNote automatically saves documents for you.
