Section 1.4 Use Cursor Routing Keys to Edit

Navigation within a document will be important when you wish to read and edit your work. You can quickly jump from the top to the end of your document. With simple commands, it is possible to navigate by character, word, line, and paragraph.

Subsection 1.4.1 Tasks

  1. Check Power Status

    1. Go to Options Menu space + o

    2. To get readout of power level letter p

  2. Review Creating a New Document:

    1. Turn on the BrailleNote rocker on/off switch

    2. Enter Main Menu space + dots123456

    3. Change Speech mode Previous Thumb-key + space

    4. From Main Menu, open the Word Processor letter w

    5. Create a document letter c

    6. For list of folders space

    7. Scroll items in list space

    8. Open folder enter

    9. Writing a name of the new document (contracted UEB okay)

    10. Create document enter

    11. If speech is on, BrailleNote will say, "top of document". You may begin writing in your new document.

  3. Navigate in and Edit the Document:

    1. Delete last character backspace

    2. To move around the document without speech thumb-keys

      1. Move forward one display advance thumb-key

      2. Move backward one display back thumb-key

    3. Move to top of document space + dots123

    4. Move to end of document space + dots456

    5. Move cursor within display cursor routing keys

    6. At the cursor, you insert text by writing.

    7. Delete the character tot he left of the cursor backspace

  4. Try these additional commands within your document:

    1. Move forward by word space + dot5

    2. Back by word space + dot2

    3. Read current word space + dots25

    4. Move forward by character space + dot6

    5. Back by character space + dot3

    6. Read current character space + dots36

    7. Move by line space + dot4

    8. Back by line space + dot1

    9. Read current line space + dots14

  5. Exit document space + e

  6. The document will save automatically. You will be back in the keyword word processor menu.

  7. Complete another word processor task or return to Main Menu.
