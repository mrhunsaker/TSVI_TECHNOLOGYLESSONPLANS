Section 2.25 Change Braille Grade

Skills Addressed in this Lesson:

  • Force the Braille grade to grade 1, 2, or computer Braille.

Subsection 2.25.1 Tasks

  1. If you are writing a Braille document, you can force the Braille note to do computer Braille, grade 1 Braille, or grade 2 Braille.

  2. Create a new Braille document as usual.

  3. Write a Braille sentence telling what you like to search for on the Google web site.

  4. Then force computer Braille in order to correctly write the web site. Switch to computer Braille by pressing backspace with j.

  5. Then write the web address using computer Braille. For a period (a dot) use dots4 6. Do not use Braille contractions.

  6. Then switch back to grade 2 Braille. Backspace with b will go back to grade 2 Braille. Try it out.

  7. You can also switch to grade 1 Braille when you want to. Force grade 1 Braille with backspace and a together.

  8. Try switching between grade of Braille for practice.
