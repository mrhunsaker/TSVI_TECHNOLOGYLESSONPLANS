Section 2.4 Word Processing

Skills Addressed in this Lesson:

  • Create folders in Keyword.

  • Create files within folders.

  • Center a line of text.

  • Use the delete menu to delete a character, word, previous word, sentence, paragraph, or document.

  • Explore the help menu in word processing.

  • Speak the time and date.

  • Return to Main Menu.

Subsection 2.4.1 Tasks

  1. From the Main Menu open word processor space + enter or letter w, enter

  2. Create a document called "science'' in a folder called "homework''

    1. From KeyWord Menu, create a document space + enter or letter c, enter

    2. At "folder name'' prompt, type "Homework'', then press enter

    3. At the cue, "folder does not exist, create new?"

      • For yes letter y

      • For no letter n

    4. At "File Name?'' prompt, write "Science'' as the file name and press enter

    5. You will be placed in your blank document called Science.

  3. Return to Main Menu space + dots123456

  4. Create a document called "math'' in the "homework''folder

    1. From KeyWord Menu, create a document space + enter or letter c, enter

    2. At "folder name'' prompt toggle to homework folder space + enter

    3. At "File Name?'' prompt, write "Math'' as the file name and press enter

    4. You will be placed in your blank document called Math.

  5. Return to Main Menu space + dots123456

  6. Give your student the time necessary to create other folders and files as desired.

  7. Open the "Math'' document

    1. From KeyWord Menu, open a document space + enter or letter o, enter

    2. Toggle options to desired folder space, enter

    3. Toggle options to desired file space, enter

  8. When in the document, instruct your student to write:

    1. Their name, then enter

    2. The date, then enter

    3. The word "Practice''

    4. Presenter twice to place a blank line after "Practice''

    5. Center text command enter + c

    6. Write "Math''

    7. exit the document space + e

  9. Using the Delete Menu

    1. Have student a document and write five sentences about a pet or favorite animal.

    2. Access delete menu space + d

    3. Toggle options space, enter or first letter, enter

      • To delete current character backspace + dots36

      • To delete current word backspace + dots25

      • To delete previous word backspace + dot2

      • To delete to the end of the sentence (asks for confirmation) backspace + dots14

      • To delete to the end of the paragraph (asks for confirmation) backspace + dots2356

      • To delete to the end of the document (asks for confirmation) backspace + dots456

  10. Open the help menu in KeyWord space + h

  11. When in the help menu, the options are:

    • edit commands

    • review commands

    • braille thumb-key commands

    • miscellaneous commands

  12. To speak the time from where you are in keyword enter + t

  13. To speak the date enter + d

  14. Return to document advance thumb-key

Subsection 2.4.2 Keyword Help Menu

  1. Edit Commands

    1. Delete character under cursor backspace + dots36

    2. Delete word under cursor, press space with dots2 and 5.

    3. Delete previous word, press backspace with dot2.

    4. Delete to end of sentence, backspace with dots1 and 4.

    5. Delete to end of paragraph, press backspace with dots2 3 5 6.

    6. Delete to end of document, press backspace with dots4 5 6.

    7. Center a line, press enter with c.

    8. Find and replace text string, press backspace with f.

    9. Change or review the layout, press space with dots2 3 4 6, l or backspace with l.

    10. Review or change page settings, press space with dots2 3 4 6 p or backspace with p.

    11. To enter a Unicode character, press backspace with dots3 5.

  2. Review Commands

    1. Go forward reading, space+g.

    2. Top of file, space + dots123

    3. Bottom of file, space + dots456

    4. Find text string space + f

    5. Find next occurrence of text string last searched space +n

    6. To go back a character, space + dot3

    7. To hear the current character space + dots36

    8. To move to the next character space + dot6

    9. To move back a word space + dot2

    10. Hear current word space + dots25

    11. To move forward a word space + dot5

    12. To move back a sentencespace + dot1

    13. To hear the current sentence space + dots14

    14. To move forward a sentence space + dot4

    15. To move to the previous paragraph space + dots23

    16. Hear current paragraph space + dots2356

    17. To move forward a paragraph space +dots56

    18. To determine the current reading mode space + m

    19. To change the reading modespace + m repeatedly

      • Sentence

      • Paragraph

      • Line

      • Columns

    20. Announce key names space + w

    21. To query the cursor position space + dots156

    22. Go to any page, line, or column space + dots126

    23. To enter or leave review-only mode space + x

  3. Braille Thumb-key Commands

    1. To advance the display by one widthadvance thumb-key

    2. To move the display back by one widthback thumb-key

    3. To start the display advancing automatically space + dots12456

    4. To control the speed previous thumb-key or next thumb-key

    5. To stop previous thumb-key + next thumb-key

    6. To move the display back a word previous thumb-key + back thumb-key

    7. To move the display on a word previous thumb-key + advance thumb-key

    8. To turn speech on or off previous thumb-key + space

    9. To turn the Braille display on or off next thumb-key + space

    10. To route the cursor to the beginning of the braille display backspace + advance thumb-key

    11. To cycle among the four keyboard modes previous thumb-key + next thumb-key

  4. Miscellaneous Commands

    1. Block commands menu space + b

    2. Spelling checker space + dots16

    3. Save file space + s

    4. Abandon edit backspace + q

    5. Open another document space + dots1256

    6. To review or change the current presentation style backspace + s

    7. To insert a field or control an address list template space + dots2346 or backspace + enter + dots2346

Subsection 2.4.3 Quiz

  1. Require your student to enter the Homework folder. Create a journal file. Your student may write a journal entry at this time.

  2. Instruct your student to create a new folder and file for you.

  3. From within a keyword document, the student should show you how to speak or display the time and date.

  4. At the top of the document, the student can show you how he or she is able to center the top line.

  5. Watch and listen as the student shows you how to use the help menu in keyword.

  6. Your student should explain how to use the delete menu and what items can be deleted in this manner.
