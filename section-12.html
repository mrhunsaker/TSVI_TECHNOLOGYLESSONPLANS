Section 1.12 Set up WiFi

The BrailleNote has a web browser application called Key Web. As with web browsers such as Internet Explorer and Firefox, Key Web can be used to read and interact with web pages. You can conduct a search using google, check your Facebook page, or download a book from Bookshare. There are many ways to use Key Web for completing Internet tasks.

The first job to be done is to set up the wireless connection. Students may help with this task so that setting up a wireless connection can be completed in other environments more independently. Students may want to use a wireless connection at home or in a public library.

Subsection 1.12.1 Tasks

Below is a sample of how to connect the BrailleNote to wireless Internet that is an open connection with no password required. Setting up a wireless or LAN connection varies depending on the settings that are in place. For school connections, the IT staff may be able to help. It is often helpful to have HumanWare tech support on the phone while working with IT staff. Arrange times with all concerned ahead of time if possible.

Sample of Setting Up a Wireless Connection:

  1. Turn BrailleNote on.

  2. Open Options Menu space + o

  3. Connectivity Options letter c

  4. Setup Options for Wireless Internet letter w

  5. "Wireless Ethernet on? Currently no" is displayed.

  6. Toggle on letter y

  7. Scan for Available Connections letter s

  8. Find your connection in list space / backspace, enter

  9. "Entry list for connection configurations"

  10. Go to first field down arrow

  11. "Connection configuration name?" is displayed along with the name of your connection name.

  12. Move to next field advance thumb-key

  13. "Obtain an IP address automatically? Yes" is displayed.

  14. Move to next field advance thumb-key

  15. "use a proxy server for this connection? No" is displayed.

  16. If I wanted the option to say yes letter y

    • if I did change to "yes" other fields and choices would become available.

  17. Context dependent help space + h

  18. When you finish with the settings, exit space + e

  19. "Add this record?" letter y for yes.

  20. "Wireless Ethernet menu" is displayed.

  21. "reconnect using existing configurations" space + enter.

  22. If your settings are correct, you will be connected to the wireless connection.

  23. You will not have to repeat this process

Subsection 1.12.2 Quiz

If the statement is true, write or say true. Write or say false if the statement is false.

  1. Space + o will open the options menu.

  2. Space + dots34 will toggle among choices.

  3. To exit a document, use e chord (e + space).

  4. Press back thumb-key + space to turn the speech off.

  5. In the options menu, c will open the connectivity menu.

  6. Space + dots2 5 will speak the current word when speech on or speech on request is on.

  7. to check the power status, press p chord + o.

  8. Space + f will help you find text.

  9. When writing a Key Word document, the BrailleNote will automatically save the document for you.

  10. Space + b will open the block commands menu.
