Section 3.3 Reading the Document Using Speech

To start reading through your document using speech space+g

To stop reading backspace+enter.

the line on which you stopped reading will be displayed in braille.

Continue reading through the document right panning key

To continue reading using speech

Route the cursor to your current braille location cursor routing button

Continue reading with speeech press space+g

Subsection 3.3.1 Creating a New folder on the SD card

From Main menu, open File Manager letter f, then enter

You will land in the last folder that was open.

To move among folders triangular back button

The BrailleNote Touch will say something like "alarms folder," meaning that you are in the list of folders on the internal storage drive.

To move select internal deive or external SD card triangular back button

For internal storage enter

For SD Card space+dot4 then enter.

You will now be on a list of the folders that are located on the SD card.

To create new folder space+n

Type desired name of folder, then enter.

Return to the Main Menu triangular Back button twice
