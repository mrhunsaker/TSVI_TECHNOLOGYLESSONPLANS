Section 2.25 Change Braille Grade

Skills Addressed in this Lesson:

  • Force the Braille grade to grade 1, 2, or computer Braille.

Subsection 2.25.1 Tasks

  1. If you are writing a Braille document, you can force the Braille note to do computer Braille, grade 1 Braille, or grade 2 Braille.

  2. Create a new Braille document as usual.

  3. Write a Braille sentence telling what you like to search for on the Google web site.

  4. Then force computer Braille in order to correctly write the web site. Switch to computer Braille by pressing backspace with j.

  5. Then write the web address using computer Braille. For a period (a dot) use dots4 6. Do not use Braille contractions.

  6. Then switch back to grade 2 Braille. Backspace with b will go back to grade 2 Braille. Try it out.

  7. You can also switch to grade 1 Braille when you want to. Force grade 1 Braille with backspace and a together.

  8. Try switching between grade of Braille for practice.

Subsection 2.25.2 Quiz

Say or write the letter of the best answer

  1. What will enter with i do?

    1. This gives information about the word count.

    2. Enter with i opens the insert menu.

    3. This opens the information options.

    4. Enter i opens the User Guide index.

  2. How can you force grade 2 Braille?

    1. enter with b

    2. space with b

    3. backspace with b

    4. backspace with low b

  3. How can you start the Braille display moving automatically?

    1. backspace with 1 2 4 5 6

    2. enter with 1 2 4 5 6

    3. space with 4 5 6

    4. space with 1 2 4 5 6

  4. What does backspace with j do?

    1. forces grade 1 Braille

    2. forces computer Braille

    3. forces grade 1 Braille

    4. forces a Braille table

  5. How can you check the date?

    1. enter with d

    2. space with d

    3. backspace with d

    4. enter with low d

  6. What will backspace with f do?

    1. find text

    2. find and replace text

    3. returns to file manager

    4. moves forward

  7. When writing computer Braille, how do you write a period?

    1. dots1 3

    2. dots2 5 6

    3. dots4 6

    4. none of the above

  8. When you are in a document, what will space with dots1 2 5 6 do?

    1. switch between documents

    2. find and replace text

    3. opens the spelling checker

    4. none of the above

  9. What does space with a full cell do?

    1. advances the Braille display one width

    2. opens key learn mode

    3. displays the power status

    4. returns to main menu

  10. When writing computer Braille, what is space with u, then dot4?

    1. period

    2. at @ symbol

    3. comma

    4. none of the above
