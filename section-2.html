Section 1.2 Set and Check Time and Date

There are many tools on the BrailleNote to help a student be organized at school or home. The BrailleNote can be used to check the time and date. This is done quietly and quickly with simple shortcut commands. Students and adults can increase their independence by leaving and arriving on time and meeting various time commitments.

Subsection 1.2.1 Tasks

  1. Navigate to Date and Time

    1. Begin at Main Menu.

    2. Check the time enter + t

    3. Clear the time Advance thumb-key

    4. Check the date enter + d.

    5. Clear the date display advance thumb-key

  2. Set Time

    1. Open Utilities menu letter u

    2. Open Date and Time Set letter d

    3. Toggle UK and USA date/time setting Space + dots34

    4. When "Date format? USA" is displayed enter

    5. Toggle 12 hr and 24 hr time options Space + dots34, enter

    6. Toggle time zones space+ dot34 , enter

      1. Time zome options, m=Mountain, p=Pacific, c=Central, e=Eastern

    7. "Is daylight saving time?" y for yes or n for no

    8. Write the time in 12 hour format as hours and minutes, a space, then a for a.m. or p for p.m.

    9. To move to Date menu enter

  3. Set Date

    1. Type the date as number sign month, number sign day, number sign year. For example, 09 12 2013 = dots3456,245,24 dots3456,1,12 dots3456,12,245,1,14.

    2. To delete last character backspace

    3. To accept typed date enter

    4. "Week starts on which day?" Toggle space + dots34

      1. Direct selection s=Sunday, m=Monday

    5. For context dependent help space + h

    6. Accept settings and return to Utilities enter

  4. Repeat the keyboard commands above to check the time and date.

    1. If they are not correct, try again.

    2. It may be that the "day light saving" option was set incorrectly.
