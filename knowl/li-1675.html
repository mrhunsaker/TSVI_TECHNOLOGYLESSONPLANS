Item 15.

To be an efficient Braille reader, use the BrailleNote without speech. Use the thumb-keys and the Braille display. Practice the below commands as appropriate.

  1. Use the back thumb-key to move back by one Braille display width. Use the advance thumb-key to move forward by one Braille display width.

  2. To start the display moving automatically, press space with dots1 2 4 5 6. Press the previous thumb-key to slow down the speed of automatic reading. The next thumb-key will increase the rate of automatic reading.

  3. To stop automatic reading, pres the previous and next thumb-keys at the same time.

  4. For more thumb-key commands, visit the help menu from within a book.
