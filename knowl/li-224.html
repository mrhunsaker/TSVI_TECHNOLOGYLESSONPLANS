Item 4.

Rename a File from File Manager Menu:

  1. From Main Menu, Open File Manager letter f

  2. Open Rename file submenu letter r

  3. "Rename file on drive?" is displayed. To rename a file on the BrailleNote,

    • To toggle options for SD card or Flash disk space, enter

    • To select most recently used disk enter

  4. Toggle down the list of folders to the one that contains the file you want to rename space, enter

  5. Toggle down the list of files space, enter

  6. "New name for file " is displayed. Write the new name for your file.

  7. TO accept typed name enter.

  8. "File Manager Menu" is displayed.
