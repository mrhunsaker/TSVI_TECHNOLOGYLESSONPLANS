Item.

In the middle of the front edge are three additional keys. From left to right:

  • Triangular-shaped Back button

    • Moves back one web page

    • Functions as an Escape key

    • Moves up through the directory hierarchy when using the File Manager.

  • Round Home button

    • Takes you to the Main Menu.

    • Triple tap of the Home button unloads or loads Keysoft.

  • Square context key

    • Tapping the square context key opens a context menu for any program that is running.

    • If you hold the context key down for about two seconds, a list of all running applications appears so that you can switch to any open application.

    • The applications list also contains a Clear All button which can be used to free up memory.
