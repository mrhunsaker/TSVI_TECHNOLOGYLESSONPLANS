Item 8.

To access KeySoft programs from anywhere in BrailleNote

  • To select the Word Processor backspace + enter + w

  • To select the Scientific Calculator backspace + enter + s

  • To select the Database Manager backspace + enter + d

  • To select the Games backspace + enter + g

  • To select the Planner backspace + enter + p

  • To select the Address List backspace + enter + a

  • To select the E-mail system backspace + enter + e

  • To select the Book Reader backspace + enter + b

  • To select the Internet browser, backspace + enter + i

  • To select the Chat (instant messaging), backspace + enter + c

  • To select the Media Player, backspace + enter + m

  • To select the FM Radio, backspace + enter + f
