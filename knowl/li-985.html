Item 2.

Create a document called "science'' in a folder called "homework''

  1. From KeyWord Menu, create a document space + enter or letter c, enter

  2. At "folder name'' prompt, type "Homework'', then press enter

  3. At the cue, "folder does not exist, create new?"

    • For yes letter y

    • For no letter n

  4. At "File Name?'' prompt, write "Science'' as the file name and press enter

  5. You will be placed in your blank document called Science.
