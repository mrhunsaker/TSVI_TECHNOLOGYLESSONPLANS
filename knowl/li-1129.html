Item 12.

Read the e-mail message with the following commands:

  • Go forward reading automatically space + g

  • To move back a character space + dot3

  • To read the current character space + dots36

  • To move forward by character space + dot6

  • To move to the top of the message space + dots123

  • Move to the end of message space + dots456

  • Move back a word space + dot2

  • Read current word space + dots25

  • Read forward by word space + dot5

  • Read back a sentence space + dot1

  • Read the current sentence space + dots14

  • Read forward by sentence space + dot4

  • To read back by paragraph space + dots23

  • To read the current paragraph space + dots2356

  • To read forward by paragraph space + dots56

  • Move Back or Forward one display length advance thumb-key or back thumb-key

  • Start braille display moving automatically space + dots23456

    • Control the speed of braille display previous thumb-key, next thumb-key

    • Stop the display from automatically moving previous thumb-key + next thumb-key
