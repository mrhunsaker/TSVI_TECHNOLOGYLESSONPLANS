Section 3.2 The Word Processor

At the Main menu, open the Word Processor letter w, then enter

A menu appears with options

Create letter c, then enter

Open letter o, then enter

Print letter p, then enter

Emboss letter e, then enter

Settings letter s, then enter

Settings includes several options including

Manage personal dictionary,

Spell checker language and an

Option to set the Embosser IP Address to an embosser that supports embossing to the Cloud

When done writing your document

Exit and open the Save As dialog space+e

When prompted for a file name, type it and press enter If this is the first time that you have saved a document, it will be saved in the Documents folder on the Storage drive. Open Save As dialog withotu exiting backspace+s

Following is a list of commands that you can use to move around your document

Move to top of document space+dots123

Move to bottom of document space+dots456

Move to previous character space+dot3

Move to next character space+dot6

Move to previous word space+dot2

Move to next word space+dot5

Move to previous line space+dot1 or Previous thumb-key

Move to next line space+dot 4 or Next thumb-key

Move to previous paragraph space+dots23

Move to next paragraph space+dots56

Move to beginning of line Enter+dots13

Move to end of line ENTER+dots46

Page up space+dots126

Page down space+dots345

Pan Braille to the left Left thumb-key

Pan Braille to the right Right thumb-key

Change navigation level space+t

Navigate to next navigation element space+dots46

Navigate to previous navigation element space+dots13

When you are writing UEB or English contracted Braille, the editing cursor is displayed as dots7,8. If you are editing computer Braille, the editing cursor is indicated by dot8 alone.
