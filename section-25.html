Section 2.2 Navigating Menus

Skills Addressed in this Lesson:

  • Turn the BrailleNote on or off

  • Use space to go forward through menu items

  • Use backspace to go back through menu items

  • Use the advance thumb-key to move forward through menu items

  • Use the back thumb-key to move to previous menu items

  • When on a menu item, press enter to open it

  • Use initial letters to move to and open menu items

  • Use space with e to exit a menu

  • Toggle among BrailleNote speech settings

  • Use and understand space with h to get help from anywhere

  • Use the shortcut to return to the Main Menu

Subsection 2.2.1 Tasks

  1. Instruct your student to turn on the BrailleNote using the on/off rocker switch

  2. Access Speech Options previous thumb-key + space

  3. Toggle Speech Options hold previous thumb-key, space

    • speech on

    • speech off

    • speech on request modes

  4. Toggle options within Main menu space

  5. Select desired option enter

  6. Exit current menu to return to previous space + e

  7. Direct Selection of options in Main Menu

    • To select the Word Processor letter w

    • To select the Scientific Calculator letter s

    • To select the Database Manager letter d

    • To select the Games letter g

    • To select the Planner letter p

    • To select the Address List letter a

    • To select the E-mail system letter e

    • To select the Book Reader letter b

    • To select the Internet browser, letter i

    • To select the Chat (instant messaging), letter c

    • To select the Media Player, letter m

    • To select the FM Radio, pletter f

  8. To access KeySoft programs from anywhere in BrailleNote

    • To select the Word Processor backspace + enter + w

    • To select the Scientific Calculator backspace + enter + s

    • To select the Database Manager backspace + enter + d

    • To select the Games backspace + enter + g

    • To select the Planner backspace + enter + p

    • To select the Address List backspace + enter + a

    • To select the E-mail system backspace + enter + e

    • To select the Book Reader backspace + enter + b

    • To select the Internet browser, backspace + enter + i

    • To select the Chat (instant messaging), backspace + enter + c

    • To select the Media Player, backspace + enter + m

    • To select the FM Radio, backspace + enter + f

  9. Return to Main Menu space + dots12356

  10. Turn the BrailleNote off when all tasks are complete on/off Rocker Switch

Subsection 2.2.2 Quiz

Require your student to demonstrate competency in the following tasks. This could be accomplished by using a written test or by requiring the student to use the BrailleNote to demonstrate mastery of the skills.

  1. Turn on the BrailleNote.

  2. If the BrailleNote does not say Main Menu, your student can return to it by using space with e or a full cell with space.

  3. At the Main Menu, ask the student to show you how he or she can get to the Internet prompt and open it by using only one letter. If necessary, give a verbal cue to the student to press the letter i.

  4. Instruct your student to return to the Main Menu. He or she should do this by using either e with space or dots1 through 6 with space.

  5. From the Main Menu, encourage your student to show you how to move forward and backward through the menu items using two different methods. These methods are using space and backspace keys and using the back and advance thumb-keys.

  6. From the Main Menu, instruct your student to go to the e-mail menu item and open it. This is done by pressing an e. Do not provide verbal cues unless they are necessary during this Quiz section.

  7. From the keymail menu, ask your student to show you how he or she would get help. If a reminder is required, state that space with h will get help from anywhere.

  8. Ask your student to read what is in the help file. The back and advance thumb-keys will work for now.

  9. Ask your student to return to the Main Menu. Remember, pressing e with space or a full cell with space will accomplish this task.

  10. Your student may now turn the BrailleNote off.
