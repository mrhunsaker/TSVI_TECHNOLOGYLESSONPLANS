Section 2.21 Web Braille / BARD

Skills Addressed in this Lesson:

  • Open the Web Braille web site.

  • Log into Web Braille.

  • Locate a book or magazine.

  • Save a book or magazine.

Subsection 2.21.1 Tasks

  1. Enter the Web Braille web site.

  2. You will need to log in.

  3. Use the thumb-keys to locate the name and password edit boxes. Write in your name and password and press enter to log in.

  4. On the Web Braille page, space or backspace to explore the links.

  5. For this exercise, let's download a magazine. Locate the Braille magazines link and press enter.

  6. Space and backspace through the links to locate a magazine that you are interested in. Press enter.

  7. Now move to the link of the magazine issue that you would like to download. Press enter.

  8. You will then be at the accept page. You will need to say "yes" to agree to the Web Braille conditions. Press space with dots4 6 to go to the "I Accept" button. You need the one that is best for Note Taker users. Press enter.

  9. You will then be on a page that you need to save. Press space with s to save.

  10. Space to the folder that you want to save the document in. Press enter to select the given folder.

  11. Name the magazine appropriately and press enter.

  12. Use enter with b to go back by web page. Download another magazine if you wish.

  13. If you want to download book volumes, use the same process, but enter on the volume number links to save them. After entering on the volume link you want, you will be taken to the accept page. Press space with 4 6 to move to the "best for note taker" option and press enter. Using the available Braille Book Review on the Web Braille site is a good place to start. Open the URL format version of the Braille Book Review. Then you may choose a category of books and will be able to locate books from that point.

  14. When pressing enter on a book volume, you will be taken to the next page. Press space with s to save the book volume.

  15. Save the book in a folder where you can find and open it later. Name the book volume so that you can also find it when you want to read and open it through the book reader on the BrailleNote.

  16. From Main Menu, press b for book reader. Select the folder and book volume you want. Press enter to open the book.

  17. You should be able to read the book as usual from here.
