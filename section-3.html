Section 1.3 Create a Folder Two Ways

The BrailleNote is a computer. With it, you can complete word processing tasks, use the planner to set and keep appointments, and send and receive e-mail messages. Just like on the computer, there is often more than one way to complete a task. This is true for creating folders. In this lesson, you will learn two ways of creating folders. After practicing both methods, you may decide which one is best for you.

Subsection 1.3.1 Tasks

  1. Method 1 From File Manager

    1. Begin at Main Menu space + dots123456

    2. Open File Manager

      • Move down options to File Manager space, enter

      • Direct selection of File Manager letter f

    3. Open Folder Manager

      • Move down options to Folder Manager space, enter

      • Direct selection of Folder Manager letter f

    4. Move down options to new folder space, enter

    5. "Create folder on which drive?" is displayed. "Flash disk" is displayed. The flash disk is the BrailleNote, added memory is "SD Card"

      • Move down options to SD card or Flash Disk space, enter

      • Select most recently used drive enter

    6. BrailleNote displays, "new folder name?" Begin writing a name for the folder.

    7. Accept folder name enter

    8. The folder is created and you are put back at the original Folder Manager Menu.

    9. To create another new folder from here, repeat the process

    10. To leave the Folder Manager

      • Exit and return to last level space + e

      • To return to the main menu space + dots123456

    11. Practice as needed.

  2. Method 2 From Word Processor

    1. Begin at the Main Menu space + dots123456

    2. To open Word Processor letter w

    3. Toggle options until "create a document" space, enter

    4. BrailleNote says, "folder name?" and the name of the folder you last had open is displayed

    5. To create a new folder, write the folder name

    6. To accept typed name and continue enter

    7. At Create a new folder?" prompt letter y=yes

    8. BrailleNote displays, "document to create?" You may create a document to put in your new folder, but don't have to. Your folder is created

    9. To exit back to menu Press space + e

    10. Practice as time allows or until the skill is mastered

  3. Control the Voice

    1. Turn the volume up enter + dot4

    2. Turn volume down enter + dot1

    3. Increase speech rate enter + dot6

    4. Decrease speech rate enter + dot3

    5. Increase speech pitch enter + dot5

    6. Lower speech pitch enter + dot2

Subsection 1.3.2 Quiz:

Tell what the result of each command or letter press will be.

  1. previous thumb-key with space

  2. At Main Menu, letter f

  3. enter + dot4

  4. enter + d

  5. space + 123456

  6. From Main Menu, letter w

  7. enter + dot2

  8. space + e

  9. enter + dot1
