Section 1.5 Rename and Delete from File Manager

Learning to delete and rename folders and documents can help your students to remain organized. There are a couple methods that can be used to complete these tasks. Try them all out to see what works best for you. In this lesson, you will be working from within the File Manager

Subsection 1.5.1 Tasks

You should have several folders and documents on your BrailleNote that we can use for practice in this lesson.

  1. Delete a folder from within File Manager:

    1. From Main Menu, open the File Manager letter f

    2. Open Folder Manager menu letter f

    3. In Folder Manager Menu, erase folder menu letter e

    4. "Erase folder on which drive?" is displayed.

      • To toggle options for SD card or Flash disk space, enter

      • To select most recently used disk enter

    5. Toggle to folder to erase space, enter

    6. "Erase folder name and all contained files and folders?" letter y for yes

    7. "1 folder erased. files erased" is displayed. P

    8. Return to Folder Manager Menu advance thumb-key

  2. Delete a File (Document) from the File Manager Menu:

    1. From Main Menu, Open File Manager Menu press f

    2. Enter erase file submenu letter e

    3. "Erase file on which drive?" is displayed.

      • To toggle options for SD card or Flash disk space, enter

      • To select most recently used disk enter

    4. To find folder containing file to be erased space, enter.

    5. To find file to be erased space, enter

    6. " Erase filename? sure?" letter y for yes

    7. To return to the File Manager Menu advance thumb-key

    8. Repeat process as needed.

  3. Rename a Folder from within File Manager Menu:

    1. Open File Manager letter f

    2. Open Folder Manager letter f

    3. Open rename option letter r

    4. Move down list to find folder to rename space, enter

    5. Write the new name for your folder

    6. Accept typed information enter.

    7. You will be taken back to the Folder Manager Menu.

    8. Exit Menu

      • Go back up one menu level space + e

      • Return to Main Menu space + dots123456

  4. Rename a File from File Manager Menu:

    1. From Main Menu, Open File Manager letter f

    2. Open Rename file submenu letter r

    3. "Rename file on drive?" is displayed. To rename a file on the BrailleNote,

      • To toggle options for SD card or Flash disk space, enter

      • To select most recently used disk enter

    4. Toggle down the list of folders to the one that contains the file you want to rename space, enter

    5. Toggle down the list of files space, enter

    6. "New name for file " is displayed. Write the new name for your file.

    7. TO accept typed name enter.

    8. "File Manager Menu" is displayed.

  5. Hotkeys:

    • Open File Manager from anywhere on the BrailleNote enter + backspace + f

    • Open Word Processor from anywhere on the BrailleNote enter + backspace + w
