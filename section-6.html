Section 1.6 Rename and Delete from List

In the last Lesson, you learned to delete folders and files from within the File Manager. You also learned to rename folders and files by going through the BrailleNote File Manager. These tasks can be done from within a list when in the BrailleNote Word Processor. After this lesson, you can decide which method is most efficient for you.

Subsection 1.6.1 Tasks

  1. Rename a Folder or Document:

    1. Begin at Main Menu. Open Word Processor letter w

    2. Open a Document

      • Toggle to find "Open a Document" space, enter

      • Directly select "Open a Document" letter o

    3. Find folder you want to rename space

    4. Rename the folder in focus, press backspace + r

    5. BrailleNote displays, "new name for "foldername".

    6. Write the new name for the folder.

    7. To accept the new name enter.

    8. The new name for the folder will be displayed.

    9. Practice as needed.

    10. The process is the same for renaming a file.

      • Focus on the document you want to rename from within the list.

      • Start rename process backspace + r

  2. Erase a Folder or Document:

    1. Begin at Main Menu. Open Word Processor letter w

    2. Open a Document

      • Toggle to find "Open a Document" space, enter

      • Directly select "Open a Document" letter o

    3. Find folder you want to erase space

    4. This process can only be completed from within a list.

    5. When the folder you want to delete has focus backspace + dots14.

    6. The process is the same for erasing a file.

      • This process can only be completed from within a list.

      • Focus on the document you want to erase from within the list.

      • Delete the file backspace + 14

  3. Practice as needed.

Subsection 1.6.2 Quiz

Write the letter of the best answer choice.

  1. What is the command to rename a document?

    1. enter + r

    2. backspace + r

    3. space + r

    4. letter r

  2. Space + dots1 2 3 4 5 6 will .

    1. reset the BrailleNote.

    2. back up by level.

    3. return to Main Menu.

    4. delete a folder.

  3. Press to toggle among the speech modes.

    1. previous + next

    2. space with a full cell

    3. Previous with space

    4. next with space

  4. How can you check the time?

    1. space + t

    2. backspace + t

    3. previous with space

    4. backspace with t

  5. Enter with dot1 will .

    1. turn volume down

    2. decrease rate

    3. make the pitch lower

    4. make the volume louder

  6. To decrease the rate, press .

    1. backspace + dot3

    2. enter + dot6

    3. enter + dot3

    4. enter + dot2

  7. Enter + dot2 will .

    1. make the pitch higher

    2. make the pitch lower

    3. make the rate lower

    4. make the volume lower

  8. In a document, how can you move back by word?

    1. space + dot3

    2. space + dot1

    3. space + dots25

    4. space + dot2

  9. Space with dot6 in a document will

    1. read the current character.

    2. read and move back by character

    3. read forward by character

    4. read forward by word

  10. O with space, then letter p will

    1. check the power status.

    2. protect the BrailleNote.

    3. move to previous word.

    4. perform a reset.
